Quiz Bulking – Easy Step-by-Step Guide
This feature lets you add many questions at once using a CSV file. Follow these steps:
  1. Download a template that matches your question type (Multiple Choice, True/False, or Essay). You can also pick “All Types”.
  2. Open the CSV (Excel, Google Sheets, or similar). Keep the columns: type, question, choiceA, choiceB, choiceC, choiceD, correct.
    • For Multiple Choice, fill choices A–D and set correct to A, B, C, or D.
    • For True/False, leave choices blank and set correct to TRUE or FALSE.
    • For Essay, just write the question. Choices can be blank.
  3. Save your file as CSV. (In Excel: File → Save As → CSV.)
  4. Upload the CSV using “Preview & Validate”. The system will check your rows.
  5. Review the table:
    • Rows marked OK are valid.
    • Rows with Error need fixing in your CSV (hover to see the reason).
    • You can Remove any valid row you don’t want to add.
  6. Click “Add valid questions to quiz”. Only valid and not-removed rows will be added.
  7. You may still edit the questions after adding them.
